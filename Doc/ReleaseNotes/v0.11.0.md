## What's New

### English
* Torrent list UI polish: centered the category filter, unified pill sizes across iOS/macOS, and replaced category text tags in rows with compact category icons.
* Torrent row actions pill now matches the same pill height as filters/controls for a more consistent layout.
* Server configuration: improved base URL validation (empty host now correctly fails).

### Русский
* Полировка UI списка торрентов: фильтр категории теперь по центру, размеры капсул унифицированы на iOS/macOS, а текстовые теги категории в строках заменены на компактные иконки.
* Капсула действий в строке торрента приведена к той же высоте, что и капсулы фильтров/контролов, для визуальной консистентности.
* Настройка сервера: улучшена валидация базового URL (пустой host теперь корректно считается ошибкой).
